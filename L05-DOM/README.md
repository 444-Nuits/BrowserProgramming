# Lecture 05 – Victor Bosshard Portfolio

## Features
- **Skills section** – 8 skills shown as interactive cards
- **Projects section** – 2 project entries with descriptions and GitHub links
- **Theme toggle** – Switch between dark (default) and light mode
- **localStorage persistence** – Theme choice saved and restored on reload
- **Last updated** – Auto-generated date shown in the footer

## How to test
1. Open `index.html` in a browser
2. Click **Toggle Theme** (top-right) to switch between dark and light mode
3. Reload the page — theme persists via `localStorage`
4. Check the footer for the auto-generated last-updated date
5. Open DevTools console to see click count logs
